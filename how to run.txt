
To learn about the Jupyter Notebook, see https://www.datacamp.com/community/tutorials/tutorial-jupyter-notebook

For the following to run, Anaconda must be installed (see https://courses.edx.org/courses/course-v1:BerkeleyX+Data8.1x+1T2018/courseware).

Run the Anaconda Prompt from a directory from which you can navigate down to the *.ipynb files.

>jupyter notebook
Should open browser tab with the URL http://localhost:8888/tree

Open the local (git-controlled) PlaylistsAnalyzer.ipynb file.
